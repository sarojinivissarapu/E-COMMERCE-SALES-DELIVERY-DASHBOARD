# E-Commerce Business Intelligece Dashboard
This Projrct Demonstrates how to clean, transform,and analayse e-commerce sales data using Excel + Power Query.
It Feaures KPIs(Total Revenue, Average Order Value, On-Time Delivery%,Top Customer) and interactive charts for regional sales, Product revenue, monthly trends, Payment methods, and delivery performance.
## Tools Used
- Microsoft Excel
- Power Query
- PivotTables & PivotCharts
## Insights
- Soth region generated the highest sales
- Tablets and phones contributed most to revenue
- On-time delviery rate ~60%
- Rahul was the top custoemr with $7,800 in Purchases
